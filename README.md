# Hotel

Sistema de información para la administración de un hotel, hecho según los requerimientos del propietario.

Permite almacenar toda la información referente a los huéspedes, así como las habitaciones, pudiendo visualizar de forma gráfica, a través de los distintos colores, el estado de cada una.

- C#
- SQLite
- .NET Framework 2.0.
